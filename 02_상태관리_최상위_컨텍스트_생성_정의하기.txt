wrap 폴더 안에

1. context 폴더 만들기
2. context 폴더 안에 컨텍스트 파일 만들기  StateContext.jsx
3. createContext 리액트에서 임포트 한다. 
4. 파일이름과 같은 컨텍스트 변수를 생성한다. 
   기본값으로 null 내보내기한다. 익스포트

5. 최상위 컴포넌트 위치에 컨텍스트를 임포트하고 
   모든 컴포넌트를 컨텍스트 프로바이더로 감싸준다
6. 컨텍스트 프로바이더에 value={{프롭스를 설정한다.}}

   내려받아 사용할 수 있다.
   import { StateContext } from "./context/StateContext";
   const {프롭스} = React.useContext(StateContext);
