* 설치(install) 하기

- 최신버전설치
npm  i  redux;

- 버전선택설치
npm  i  redux@4.1.2;

- 패키지 1개씩 설치하는 방법
npm  i  redux@4.1.2;
npm  i  react-redux;

- 패키지 2개 설치하는 방법
npm  i  redux@4.1.2  react-redux;
