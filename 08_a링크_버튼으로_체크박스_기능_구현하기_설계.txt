a링크 버튼으로 체크박스 기능 구현하기 설계
클릭 이벤트
아규먼트 2개 전달(e, idx) => 매개변수 2개 받는다(e, idx)

.on 클래스 를  3항연산자 사용 해당버튼 체크 기능 수행하기위해 상태변수 배열변수 를 사용
초기값 모두 false
arr = [false, false, ... , false]
arr[0]?'on':'' => 체크해제

해당버튼[0] 클릭하면 
클릭하면 해당 버튼에만 true로 변경
arr = [true, true, ... , false]
arr[0]?'on':'' => 체크설정


배열 초기값은  각 버튼 갯수에 따라 false 초기값 갯수가 다르다
